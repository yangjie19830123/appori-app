# 英雄之路 · Heroes v0.1

参照英雄无敌 3 的单人战棋闯关游戏。当前版本：**第一章 · 训练之路**（5 关）。

部署方式跟 calendar / shift / wsi / uno 完全一致：把 `heroes/` 文件夹丢进 `appori-app/app/`。

## 操作步骤

### 1. 把 heroes 文件夹拖进 appori-app

```
C:\Users\YangJie\appori-app\app\heroes\
```

完成后 `app/` 下应该是：

```
appori-app\app\
├── calendar\
├── kanji\
├── kakeibo\
├── shift\
├── wsi\
├── zorox\
├── uno\
├── heroes\        ← 新增
├── layout.tsx
└── page.tsx
```

### 2. 不需要装新依赖

跟 calendar/wsi 一样，纯前端。**不像 UNO 那样需要 Pusher 和 Upstash**——所有数据都存在玩家自己的浏览器 localStorage 里。

### 3. 部署

```powershell
cd C:\Users\YangJie\appori-app
vercel --prod
```

完成后访问 `https://www.appori.app/heroes`。

## 第一版包含什么

**核心系统**：

- 英雄系统：凯瑟琳，4 项属性（攻/防/法力/法力值），等级 1-10
- 升级系统：通关获得经验，升级后弹窗三选一加点（HMM3 风格）
- 法术系统：3 个法术
  - ✨ 治疗（4 法力）：恢复友军 HP
  - 💨 加速（3 法力）：友军移动+2，持续 2 回合
  - ⚡ 神圣审判（6 法力）：对敌人造成无视护甲的伤害
- 兵种系统：3 种兵相互克制
  - 🛡️ 步兵：血厚、克骑兵
  - 🏹 弓箭手：远程、克步兵
  - 🐎 骑兵：高速、克弓箭手
- 战斗系统：
  - 5×8 矩形战场
  - 单位按速度排序行动
  - 简易 AI（找射程内目标 → 攻击 / 朝最近敌人移动）
  - 倍速 ×1 ×2 ×4 + 暂停 + 单步
  - 胜负判定 + 30 回合平局保护

**第一章 5 关**：

| 关卡 | 名字 | 难度 |
|---|---|---|
| 1-1 | 第一课 | ⭐ 教学关 |
| 1-2 | 树林伏兵 | ⭐⭐ 简单 |
| 1-3 | 骑兵突袭 | ⭐⭐ 中等 |
| 1-4 | 三方混战 | ⭐⭐⭐ 中等 |
| 1-5 | BOSS：哥布林酋长 | ⭐⭐⭐⭐ 必须用法术 |

**3 星评分**：

- ⭐ 通关
- ⭐⭐ 通关 + 我方损失 ≤ 30-50%（按关卡难度）
- ⭐⭐⭐ 通关 + 我方损失 ≤ 10-15%

**存档**：

- 自动保存到浏览器 localStorage
- 主页底部有"重置存档"按钮（开发期方便测试）

## 故障排查

| 症状 | 原因 |
|---|---|
| `/heroes` 404 | 拖文件夹没拖对位置 |
| 战斗页一直加载 | 动态路由参数解析问题，看 Console |
| 字体渲染失败 | 极小概率 Vercel 拉 Google Fonts 失败，重新部署 |
| 进度丢失 | 玩家清了浏览器数据，无法恢复（v0.1 只本地存档） |
